# AI Chat Project Rules

## Frontend

* Vue3
* Vite
* TypeScript
* Pinia
* Axios
* Element Plus
* TailwindCSS

## Backend

* FastAPI
* PostgreSQL
* SQLAlchemy
* JWT
* uv
* langchain

## Rules

* 使用TypeScript
* 使用组合式API
* 使用script setup
* 所有接口必须有类型定义
* 不允许any
* 不允许在 Router 中编写业务逻辑
* Service 层负责业务逻辑
* App.vue是根组件只包裹router-view 或核心视图

## Git Rules

完成一个功能后提醒执行：

git add .
git commit
